# IoT-Based Face Recognition System (ESP32-CAM)

## Description
This project implements a face recognition system using the ESP32-CAM module. It captures real-time images and performs basic recognition logic, demonstrating embedded AI and IoT-based system design.

## Technologies Used
- ESP32-CAM
- Arduino IDE
- Embedded C / C++
- Camera Module

## Components Used
- ESP32-CAM module
- Camera
- Power supply
- WiFi connectivity

## Features
- Real-time image capture
- Face detection capability
- Wireless data transmission using WiFi
- Embedded system integration

## Working
1. ESP32-CAM captures live video/images  
2. Processes image data using embedded logic  
3. Detects and identifies faces  
4. Sends output or triggers response
